Static Site Generator

Credit to Boot.Dev for project guidance.

Thanks for visiting my Static Site Generator demo, built with Python! After running the program, you'll see the initially empty public folder gets filled with files by the 9-second mark. By the 13-second mark, the local server loads the website, rendering the content. The website files in the public folder are generated from markdown in the content folder and media from the static folder.
